# ESX & QB-Core Time and Weather Script

This FiveM resource allows server admins to adjust the server time and weather using chat commands.

## Features
- Set server time
- Freeze/unfreeze server time
- Set server weather
- Freeze/unfreeze server weather
- Display available weathers with `/weathers`

## Commands
- `/settime [hour] [minute]` - Set server time
- `/setweather [weather]` - Set server weather
- `/f` - Freeze time and weather
- `/uf` - Unfreeze time and weather
- `/weathers` - Display available weathers

## Installation
1. Download the files
2. Place them in your FiveM server resource folder
3. Add `start esx_timeandweather` to your server.cfg
